# Monkey
Monkey is an Interpreter written in Go

## Setup
To setup this project you need to have installed 
golang on your machine and a basic understanding of 
linguistics hahaha

## Example
```monky
let a = 5;
let b = 6;

let add = fn(x,y) {
  x + y;
}

let result = add(a,b)
```
> The result will be 11
